# Barcodes #

Windows install:

Download python from:
https://www.python.org/ftp/python/3.5.3/python-3.5.3.exe

Install with checked:
Add Python 3.5 to PATH

Download pip:
https://bootstrap.pypa.io/get-pip.py

Install by command:
python get-pip.py

Install libraries:
pip install pillow
pip install pdf417gen
pip install django

Run:
runserver.bat

Web address at browser:
http://127.0.0.1:8000
